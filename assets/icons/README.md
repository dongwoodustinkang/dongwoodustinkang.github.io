Icon assets for awards/footer can be SVG or PNG.

Store files in this folder:
- `assets/icons/award-scholarship.svg` or `assets/icons/award-scholarship.png`
- `assets/icons/award-contest.svg` or `assets/icons/award-contest.png`
- `assets/icons/github.svg` or `assets/icons/github.png`
- `assets/icons/threads.svg` or `assets/icons/threads.png`
- `assets/icons/linkedin.svg` or `assets/icons/linkedin.png`
- `assets/icons/google-scholar.svg` or `assets/icons/google-scholar.png`

Behavior:
- CSS is set to try SVG first, then PNG fallback.
- If files are missing, default circle placeholders still appear.
